# 🕵️ AI-Based Lie Detector System

## 📌 Overview

The AI-Based Lie Detector System is a machine learning project designed to analyze facial expressions and behavioral cues from video input to predict whether a statement is truthful or deceptive. The system combines computer vision, deep learning, and video processing techniques to provide real-time lie detection insights.

This project demonstrates the application of Artificial Intelligence, Machine Learning, and Computer Vision in human behavior analysis.

---

## 🚀 Features

* Real-time video analysis
* Facial feature detection using OpenCV
* Deep learning-based prediction model
* Automated truth/deception classification
* User-friendly prediction workflow
* Modular and scalable project structure

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Rushi10082005/Lie-Detector.git
cd Lie-Detector
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

```bash
python app.py
```

or

```bash
python main.py
```
